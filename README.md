# 🤖 ClashManage - Bot Discord

## Description
**ClashManage** est un bot Discord conçu pour les joueurs de **Clash of Clans**. Il permet de suivre les attaques et contributions dans la capitale d’un clan, ainsi que d’autres fonctionnalités utiles pour gérer et organiser votre clan directement depuis Discord.  

Le bot est simple à configurer et entièrement personnalisable selon vos besoins.

---

## 🛠️ Fonctionnalités
- Suivi des attaques dans la capitale du clan
- Suivi des contributions des membres
- Commandes interactives pour consulter les stats
- Notifications automatiques pour les mises à jour importantes

---

## ⚙️ Installation
1. Cloner ou télécharger ce dépôt.
2. Installer les dépendances :

```bash
npm install
```

## Utiliser le Bot Discord

- Dans le fichier ```.env```, rajouter votre clé API Supercell (https://developer.clashofclans.com/#/), et ajouter votre TAG_DE_VOTRE_CLAN.
- Démarrer le Bot : 

```bash
node main.js
```
